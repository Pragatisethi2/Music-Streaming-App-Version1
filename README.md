## Getting Started

### Prerequisites

- Python 3.x
- pip


### Installation

1. Clone the repository.

```
git clone https://github.com/ShubhamYadav9416/Ticket_show_app.git

```

2. Install the requirements 
 
```
pip install -r requirements.txt

```

3. Run the application. 

```
python main.py
```
